Privacy Policy - LunaUx Decompiler Bot

Effective Date: November 23, 2025

1. Introduction

This Privacy Policy explains how LunaUx Decompiler Bot. The bot collects, uses, processes, and protects your information when you use our Discord bot service. We are committed to protecting your privacy and handling your data responsibly.

2. Information We Collect

2.1 Automatically Collected Data

  • Discord User IDs: Used for authorization, access control, and command execution
  • Discord Server IDs: Used to identify where the bot is installed and configured
  • Command Usage: Records of which commands are executed (for analytics and debugging)
  • Timestamps: When commands are used and files are processed
  • Error Logs: Technical information when errors occur during processing

2.2 User-Submitted Content

  • Luau Bytecode Files: Files uploaded for decompilation or disassembly
  • Command Parameters: Options and settings provided with bot commands
  • File Metadata: File names, sizes, and formats of uploaded content
Important: We do NOT collect or store message content, personal conversations, server messages, or any communication not directly related to bot commands.

3. How We Use Your Information

3.1 Primary Purposes

  • Service Delivery: Processing decompilation and disassembly requests
  • Access Control: Managing user permissions and command restrictions
  • Bot Functionality: Executing commands and returning results
  • Performance Monitoring: Tracking processing times and resource usage

3.2 Secondary Purposes

  • Service Improvement: Analyzing usage patterns to enhance features
  • Debugging: Identifying and fixing technical issues
  • Security: Detecting abuse, spam, or unauthorized access attempts
  • Rate Limiting: Preventing service abuse and ensuring fair usage

4. Data Storage and Retention

File Processing Policy: All uploaded files are processed in memory or temporary storage and are automatically deleted immediately after decompilation is complete. We do not maintain permanent copies of your submitted files.

4.1 Retention Periods

  • Uploaded Files: Deleted immediately after processing (typically within seconds)
  • Command Logs: Retained for up to 30 days for debugging purposes
  • User IDs: Stored only while the bot is active in your server
  • Error Logs: Retained for up to 90 days for technical analysis
  • Access Control Lists: Maintained as long as necessary for bot operation

5. Data Sharing and Disclosure

5.1 We Do NOT Share Your Data With:

  • Third-party advertisers or marketing companies
  • Data brokers or analytics services
  • Other Discord bots or services
  • Any commercial entities for profit

5.2 Limited Disclosure Scenarios

We may disclose information only in the following circumstances:

  • Legal Requirements: If required by law, court order, or government regulation
  • Discord ToS Violations: Reporting violations to Discord Trust & Safety
  • Service Protection: Preventing fraud, abuse, or security threats
  • With Your Consent: When you explicitly authorize data sharing

6. Data Security

6.1 Security Measures

  • Encrypted Connections: All data transmission uses Discord's secure protocols
  • Access Controls: Restricted access to bot systems and databases
  • Temporary Processing: Files processed in isolated, temporary environments
  • Automatic Deletion: Systematic removal of processed files
  • Regular Audits: Periodic security reviews and updates

6.2 Limitations

While we implement robust security measures, no system is 100% secure. We cannot guarantee absolute security of data transmitted over the internet or stored electronically.
